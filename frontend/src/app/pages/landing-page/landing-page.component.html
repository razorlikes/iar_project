Welcome in the Integration Architectures Template Webapp!

You can add your own content to this page.
You will find its files at: frontend/src/app/pages/landing-page/

How to add new pages?

The pages, which are handled by the angular router, are basically angular components. So first of all, we need to add a new component.
To preserve the syntactic structure of the project, you may put this component inside the directory frontend/src/app/pages/ .
