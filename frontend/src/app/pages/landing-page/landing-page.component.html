Welcome in the Integration Architectures Template Webapp!

You can add your own content to this page.
You will find its files at: frontend/src/app/pages/landing-page/

How to add new pages?

  1. Create a new Component

    The pages, which are handled by the Angular router, are basically Angular components. So first of all, we need to add a new component. (In IntelliJ, you can simply right-click the target directory and select New > Angular Schematic... > component . Otherwise you can use Angular CLI and type ng generate component [component-name]. It will generate a component in the current directory.)
    To preserve the syntactic structure of this project, you may place this component inside the directory frontend/src/app/pages/ .

  2. Register Component in the Angular Router

    After you created your new component, its needs to be registered in the Angular router, to make it accessible via an URL.
    Please edit frontend/src/app/app.routing.ts to register your component.

  3. (Optional) Add Menu-Button

    The menu-buttons are specified in an array in frontend/src/app/components/menu-bar/menu-bar.component.ts .
    Simply add your new page there with its path from the router.
